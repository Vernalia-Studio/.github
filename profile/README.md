Vernalia is a boutique studio building bespoke digital experiences for publishing and independent music — the kind of work that sits somewhere between a website and a world.

Every project is a **Fossil**: a self-contained, serverless artifact designed to outlast the trend cycle. Not a template. Not a theme. A thing built precisely for what it needs to be.

---

### THE PHILOSOPHY

The modern web optimises for frictionlessness. We build experiences that require **intention** to enter.

We work in vanilla HTML, CSS, and JavaScript, not as a constraint, but as a craft decision. The stack is lean because the work demands it. Every line of code is load-bearing.
